The following steps are mandatory to initiate our code, please follow all the steps:

1) Make sure to have all the following files inside the .rar folder:
	main.mlx , Q1_A.mlx , Q1_B.mlx , Q1_C.mlx, Q2_A.mlx , Q3.mlx.
2) Un-zip all the mentioned files to the same dirctory.
3) Open main.mlx via MATLAB (Make sure to set the MATLAB path to the extracted files folder).
4) Run the code :)
